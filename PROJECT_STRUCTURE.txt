═══════════════════════════════════════════════════════════════════
TECHNOLOGY STACK
═══════════════════════════════════════════════════════════════════

Frontend:
  • Angular 17
  • Angular Material (UI components)
  • TypeScript
  • RxJS (Reactive programming)
  • Axios (HTTP client)

Backend:
  • FastAPI (Python web framework)
  • SQLite (Metadata storage)
  • PostgreSQL / MongoDB / OpenSearch (Database options)
  • Prometheus (Metrics)
  • Pandas (Data processing)

Infrastructure:
  • Docker & Docker Compose
  • Python 3.x
  • Node.js & npm


═══════════════════════════════════════════════════════════════════
KEY FEATURES
═══════════════════════════════════════════════════════════════════

• Multi-Database Support (PostgreSQL, MongoDB, OpenSearch)
• File Processing (CSV, Excel, JSON)
• Data Visualization (Bar, Line, Pie charts with SVG)
• Dashboard with statistics & file management
• File upload with drag-and-drop & auto-upload
• Data preview tables with pagination (10 items/page)
• Authentication system (Login/Sign-up with route guards)
• Responsive design with Angular Material
